LASA {CS} Curriculum

AP Computer Science

A college level CS course teaching object oriented programming with Java.

Advanced CS

A data structures class, designed to be the next step after taking AP CS.

Digital Electronics

Learn about the hardware behind computers, transistors to microprocessors.

Fundamentals of CS

An introductory class to CS, specifically designed for beginners.

Web & Mobile Apps

Use software to build and develop applications for the web and mobile devices.

Independent Study CS

For those who have exhausted the LASA CS curriculum, and want to learn on their own.
